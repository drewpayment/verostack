How to Use Pay Runs

Glad to see you've made it here! Now, you have to tackle creating a Pay Run. Don't worry, it's much easier than it seems and you'll be a professional in no time.

You start by clicking the "+ Payroll" button on the Pay Runs header bar of this page.
Enter your start and end dates of the entire time period for which you want any current paychecks to be included.

The app allows you to designate the timeframe and doesn't restrict you. It is totally up to you to how you would like those dates to work.
